Intégration Web

Utiliser les langages HTML et CSS pour créer des pages web !

Langages utilisé

Le HTML

Le HTML permet de construire la structure de la page web et de déterminer les éléments qui composent notre page web.

Le CSS

Le CSS permet d'appliquer du style sur les éléments crées dans le HTML afin de mettre en forme la page web.

Création de page web

On construit des pages web en créant du code HTML et CSS qui est interpréte par le naviguateur web
